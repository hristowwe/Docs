##

| Команда      | Параметри                                  | Обяснение                                                                                          |
| ------------ | ------------------------------------------ | -------------------------------------------------------------------------------------------------- |
| `create`     | `id` `initial state`                       | Създава нов NFA с даден идентификатор и начално състояние [повече тук](components/create)          |
| `open`       | `filename`                                 | Отваря съществуващ файл с NFA и го зарежда в паметта [повече тук](components/open)                 |
| `save`       | `id` `filename`                            | Запазва NFA с даден идентификатор във файл [повече тук](components/save)                           |
| `list`       | Няма параметри                             | Извежда списък с наличните NFAs [повече тук](modal/list)                                           |
| `print`      | `id`                                       | Извежда преходите на NFA с даден идентификатор [повече тук](modal/print)                           |
| `empty`      | `id`                                       | Проверява дали езикът на NFA с даден идентификатор е празен [повече тук](modal/empty)              |
| `transition` | `id` `current state` `symbol` `next state` | Добавя преход от текущото състояние до следващото с дадения символ [повече тук](modal/transitions) |

:::tip
Това е обобщен списък с командите и техните параметри.
:::
